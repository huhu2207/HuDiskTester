# HuStudio Disk Tester 烂盘克星

HuStudio 烂盘克星是一个存储设备的跑分、测试工具。专治淘宝奸商、华强北奸商。
- 目标功能：

  - 测试读写速度
  - 测试实际可用容量（待完成）
  - 老化测试（循环测试，待完成）

- 编译&测试环境
  >  Visual Studio 2013 or 2015, with .NET Framework 4.5 or 4.6
  >  
  > 测试盘：WD MyPassport Ultra 1TB
  >
  > 系统：Windows 10 build 10240 RTM
